# YouTube Downloader 🎬🎵

A Python-based automation tool that downloads YouTube videos or extracts MP3 audio from either URLs or video names. It also supports batch downloading for multiple inputs.

## 🚀 Features
- Download YouTube videos using direct URLs
- Search and download videos using video names
- Extract MP3 audio from videos
- Batch processing (multiple URLs or names at once)
- Sequential downloading system
- Simple and lightweight

## 📦 Requirements

Make sure you have Python installed.
